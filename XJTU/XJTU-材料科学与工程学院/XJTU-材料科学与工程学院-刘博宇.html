个人详情

 

工作经历
西安交通大学（材料科学与工程学院）
教授 / 副教授 / 讲师（2015至今）

陕西省镁基新材料工程研究中心
副主任（2015至今）

教育经历
西安交通大学（材料科学与工程学院）
博士研究生 / 本科（2005-2015）

研究简介
研究领域为镁基材料的开发应用及相关共性基础研究，主要研究内容包括（1）采用原位多场耦合测试技术，研究镁及镁合金的塑性变形行为及其内在微观机理，为完善镁的塑性变形理论和镁合金的开发提供基础数据；（2）新型原镁冶炼和镁提纯技术及装备研发。主持国家自然科学基金和企业横向课题多项。研究成果在Science、Nature Communications、Acta Materialia等期刊发表，授权发明专利3项，荣获TMS轻金属镁分会最佳基础研究论文、国际显微学会青年科学家奖学金、陕西省科学技术一等奖（4/5）等奖项。

荣誉和获奖
2020 入选西安交通大学“青年拔尖人才”计划项目
2018 陕西省科学技术一等奖（第4完成人）

2017 陕西高校科学技术一等奖（第4完成人）

2014 国际显微学会青年科学家奖学金
2013 美国TMS学会镁合金最佳基础研究论文奖
2013 GATAN中国区博士生奖学金

2. 国家自然科学基金青年项目：镁合金中形变孪晶与析出相交互作用的原位定量透射电镜研究（20万元），2017-2019，项目负责人
3. 中国博士后科学基金面上项目：镁合金中针对形变孪晶的强韧化机理的原位电镜研究（8万元），2017-2019，项目负责人
