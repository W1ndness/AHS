# AHS Dataset
Language: [ENG](https://github.com/W1ndness/AHS#readme) | **简体中文**

## 简介

这是一个包含了来自不同学校的**学者的学术主页**的数据集。它可以用于数据挖掘和属性提取等任务。

## 内容

该数据集包含了22所国际大学，其中每所大学包含4个学院，每个学院包含至少15位学者(教师)的个人主页。
